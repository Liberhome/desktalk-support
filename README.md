# DeskTalk Support

DeskTalk is a macOS app for secure local-network chat and file sharing between nearby devices running the app.

## Quick Start

1. Connect your devices to the same Wi-Fi or local network.
2. Open DeskTalk on each device.
3. Enter the same room code on each device. The room code must be at least six characters.
4. Allow the local network permission prompt if macOS asks.
5. Send text, images, or files once the nearby devices show as connected.

## Troubleshooting

- Make sure every device is on the same network.
- Check macOS Firewall and Local Network permission settings.
- Turn off router client isolation if devices cannot discover each other.
- Use the same room code on every device.
- Received files are saved inside DeskTalk's app sandbox and can be revealed from the file attachment in the chat window.
