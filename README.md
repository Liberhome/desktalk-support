# DeskTalk Support

DeskTalk is a macOS app for secure local-network chat and file sharing between nearby Macs.

## Quick Start

1. Connect your Macs to the same Wi-Fi or local network.
2. Open DeskTalk on each Mac.
3. Enter the same room code on each Mac. The room code must be at least six characters.
4. Allow the local network permission prompt if macOS asks.
5. Send text, images, or files once the nearby Macs show as connected.

## Troubleshooting

- Make sure every Mac is on the same network.
- Check macOS Firewall and Local Network permission settings.
- Turn off router client isolation if devices cannot discover each other.
- Use the same room code on every Mac.
- Received files are saved to `Downloads/DeskTalk Received`.
