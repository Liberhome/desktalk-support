# DeskTalk Privacy Policy

Effective date: May 29, 2026

DeskTalk is designed for local-network communication between devices running the app. It does not require an account and does not use a cloud relay for messages or file transfers.

## Data Collection

DeskTalk does not collect personal data, analytics, diagnostics, advertising identifiers, contacts, location, or payment information.

## Local Data

Messages are kept only in the running app session and are not uploaded to a server by DeskTalk. Received files are saved locally inside DeskTalk's app sandbox on the receiving device and can be revealed from the file attachment in the chat window.

## Network Use

DeskTalk uses system local networking frameworks to discover and communicate with nearby devices on the same local network after you set a room code.

## Third Parties

DeskTalk does not include third-party analytics SDKs or advertising SDKs.
